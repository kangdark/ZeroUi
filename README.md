# Zero UI

Author: KangDark

Modern Roblox UI Library

## Features

- Toggle
- Button
- Slider
- Dropdown
- Notification
- Scroll Container
- Hide / Show Menu

## Usage

local ZeroUI = loadstring(game:HttpGet(
"https://raw.githubusercontent.com/kangdark/ZeroUI/main/source.lua"
))()

local Window = ZeroUI:CreateWindow({
    Name = "My Hub"
})
